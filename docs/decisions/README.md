# Decision Snapshots

意思決定の本文を格納するディレクトリ。

## 命名規約

`d-YYYY-MM-DD-short-kebab.md`

- `YYYY-MM-DD`: 決定日
- `short-kebab`: 内容を表す短いケバブケース

## 必須セクション

各 Decision ファイルには以下のセクション（`## <名前>`）を必ず含めること:

1. Decision-ID
2. Context
3. Rationale
4. Alternatives
5. Impact
6. Verification
7. Supersedes
8. Inputs Fingerprint

テンプレート: `_template.md`
